To run this assembly-based Tic Tac Toe game on your system follow these steps:

1. Ensure you have a DOSBox assembler installed on your computer
2. Open the project directory abd assemble the source code using TASM.game
3. Link the generated object file using Tlink or another linker( Tlink game)
4. Run the compiled executable (game.exe)
   Enjoy the game!
